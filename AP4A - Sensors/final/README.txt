===== PROJET AP4A [A22] =====

Ce projet à pour but de simuler la récolte de différentes données procurés par les sensors dans un avion. Pour effectuer cela nous avons crées différentes classes.

==Sensor==

Cette classe template à pour but la gestion du sensor. On crée une classe fille pour chaque type de sensor [Humidity, Light, Pression, Temperature]. 

==Server==

Cette classe gère l'enregistrement des différentes de données. C'est à dire l'écriture des données dans les différents fichiers et l'écriture dans la console.

==Scheduler==

Cette classe gère les requêtes de l'utilisateur et s'assure de la gestion des différents sensors mais aussi de l'objet server.
